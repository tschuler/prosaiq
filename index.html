Welcome to PROsaiq


These pages describe a pilot project conducted by Thilo Schuler under the supervision of Andrew Miller while at the Department of Radiation Oncology, Illawarra Cancer Care Centre, Wollongong Hospital, Wollongong NSW Australia.

The project goal was to build a pilot system demonstrating the feasibility of a low cost electronic patient-reported outcome (ePRO) system integrated with the existing electronic medical record (EMR) for routine care data collection.

The project is described in detail in an open-access JROI article (please use for citing): coming soon...

The following pages provide a concise overview:
  • Background - outlines what motivated us to create PROsaiq
  • Online Demo - helps to quickly grasp the PROsaiq's main ideas and functionality
  • Use Cases - shows possible application areas of PROsaiq or similar software
  • Technology - includes links to re-used software components and PROsaiq's source code
